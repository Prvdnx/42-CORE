Color Palette
Light Mode
Primary Blue: Bright calm blue (#5B8CFF) - used for buttons, headers, active states
Background: Very light cool gray (#F5F7FA) - main page background
Card/Surface: Pure white (#FFFFFF) - containers for content
Text Primary: Almost black (#1C1C1E) - main text
Text Secondary: Medium gray (#8E8E93) - muted text, hints
Borders: Very light gray (#E5E5EA) - dividers, borders
Dark Mode
Background: Very dark gray (#1C1C1E)
Card/Surface: Slightly lighter dark (#2C2C2E)
Text Primary: White
Text Secondary: Medium gray (#98989D)
Borders: Dark gray (#38383A)
Primary Blue: Same bright blue (#5B8CFF) - maintains prominence
Feeling/Emotion Colors (same in both modes)
Happy: Bright yellow (#FFD60A)
Sad: Soft red (#FF6B6B)
Excited: Orange (#FF9500)
Calm: Light cyan blue (#64D2FF)
Anxious: Purple (#BF5AF2)

Profile Screen: After login, shows all entries and stats (default view)
Agenda Screen: Click Calendar in nav to see calendar view
New Entry: Click blue plus button to create entry
View Entry: Click any entry card to see full details
Dark Mode: Toggle with moon/sun button in header
Logout: Click logout button to return to login screen
Key Design Principles
Calming: Soft blues and grays create peaceful feel
Minimalist: Only shows necessary information
Clear hierarchy: Headers, content, actions are distinct
Consistent: Same patterns repeated (cards, buttons, spacing)
Accessible: Good contrast, readable text, intuitive layout
Emotional: Feeling system with distinct colors and icons
Smooth: All animations feel natural and responsive
