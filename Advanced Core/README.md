# Advanced Core

Advanced specialization projects focusing on mobile development and machine learning.

## Projects

### Mobile Development
- **Mobile Piscine** - Comprehensive mobile development training with React Native and Expo
  - Module 00: Introduction to React Native basics
  - Module 01: UI fundamentals and navigation
  - Module 02: API integration with weather services
  - Module 03: Advanced features (dark theme, responsive design)
  - Module 04: Authentication with Clerk and Firebase
  - Module 05: Advanced navigation with tab and calendar views

### Android Development
- **ft_hangouts** - Native Android SMS and contact management application built with Kotlin

### Machine Learning
- **ft_linear_regression** - Introduction to machine learning through linear regression implementation
  - Gradient descent algorithm
  - Data visualization
  - Price prediction model

## Skills Developed
- Cross-platform mobile development (React Native/Expo)
- Native Android development (Kotlin)
- Mobile UI/UX design
- API integration and data fetching
- Authentication and secure storage
- Machine learning fundamentals
- Data analysis and visualization
