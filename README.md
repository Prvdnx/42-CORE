# 42-CORE

Welcome to my 42 School core curriculum repository. This repository tracks my progress and projects through the 42 program.

## Repository Structure

The repository is organized into two main sections:

### 1. Common Core
Foundational projects of the 42 curriculum.

#### Projects in Common Core:
- **[CPP Modules:](https://github.com/Prvdnx/42-CPP_MODULES)** C++ Piscine projects.
- **[FdF:](https://github.com/Prvdnx/42-FdF)** 3D Wireframe Rasterizer.
- **[Inception:](https://github.com/Prvdnx/42_Inception)** System Administration and Docker.
- **[NetPractice:](https://github.com/Prvdnx/42-NetPractice)** Networking exercises.
- **[Philosophers:](https://github.com/Prvdnx/42-Philosophers)** Concurrency with threads and mutexes.
- **[ft_irc:](https://github.com/TanawatJukmongkol/ft_irc)** Internet Relay Chat server.
- **[ft_libft:](https://github.com/Prvdnx/42-ft_libft)** Custom C Standard Library implementation.
- **[ft_printf:](https://github.com/Prvdnx/42-ft_printf)** Recreating the printf function.
- **[get_next_line:](https://github.com/Prvdnx/42-get_next_line)** Reading a line from a file descriptor.
- **[miniRT:](https://github.com/gs313/miniRT)** Ray Tracing with C.
- **[minitalk:](https://github.com/Prvdnx/42-minitalk)** Small client-server communication using signals.
- **[push_swap:](https://github.com/Prvdnx/42-push_swap)** Stack sorting algorithm.
- **[transcendence:](https://github.com/bpassael/transcendence)** Real-time multiplayer Pong web application.

### 2. Advanced Core
Advanced specialization projects.

#### Projects in Advanced Core:
- **[Mobile Piscine:](<Advanced Core/MOBILE PISCINE/>)** Developing mobile applications.
- **[ft_hangouts:](<Advanced Core/ft_hangouts/>)** An Android SMS and contact management application.
- **[ft_linear_regression:](<Advanced Core/ft_linear_regression/>)** Introduction to machine learning concepts.
