# Common Core

Foundational projects of the 42 curriculum covering essential programming concepts, algorithms, and system administration.

## Projects

### C/C++ Programming
- **CPP Modules** - C++ Piscine covering OOP, templates, STL, and advanced C++ concepts
- **ft_libft** - Custom implementation of C Standard Library functions
- **ft_printf** - Recreation of the printf function with format specifiers
- **get_next_line** - Function to read lines from file descriptors
- **minitalk** - Client-server communication using UNIX signals
- **push_swap** - Efficient stack sorting algorithm implementation

### Graphics & Algorithms
- **FdF** - 3D wireframe rasterizer with isometric projection
- **miniRT** - Ray tracing engine built with C

### Concurrency & Networking
- **Philosophers** - Dining philosophers problem using threads and mutexes
- **NetPractice** - TCP/IP networking configuration exercises
- **ft_irc** - Internet Relay Chat server implementation

### System Administration
- **Inception** - Multi-container Docker infrastructure with NGINX, WordPress, and MariaDB

### Web Development
- **transcendence** - Real-time multiplayer Pong web application

## Skills Developed
- Low-level programming in C/C++
- Object-oriented programming
- Memory management and pointers
- Concurrent programming
- Network protocols
- Graphics programming
- System administration with Docker
- Algorithm optimization

> **Note:** Some code may need to be refactored and updated.
